# NVDIA_Image-Noise-Reduction

# 🧠 Image Noise Reduction using Autoencoders (TensorFlow)

## 📌 Project Overview
This project focuses on image denoising using Deep Learning Autoencoders built with TensorFlow/Keras. The model is trained to reconstruct clean images from noisy inputs by learning efficient representations.

This project was developed during my college lab work and showcases practical implementation of GPU-accelerated deep learning.

---

## 🚀 Key Highlights
- Implemented Autoencoder architecture for image denoising  
- Trained on MNIST dataset  
- Used NVIDIA GPU (via nvidia-smi) for acceleration  
- Worked with CUDA version 12.2  
- One of the first implementations in my college lab using GPU resources  
- Demonstrated faster training compared to CPU  

---

## 🏗️ Project Workflow
1. Data Loading (MNIST dataset)  
2. Data Preprocessing (Normalization + Noise Addition)  
3. Model Building (Encoder + Decoder)  
4. Training using GPU acceleration  
5. Evaluation and Visualization  

---

## 🧠 Model Architecture
- Input Layer (Noisy Image)  
- Encoder (Feature Compression)  
- Bottleneck (Latent Representation)  
- Decoder (Reconstruction)  
- Output Layer (Denoised Image)
